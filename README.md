# EMS - Employee Management System

## Overview

EMS is a full-stack employee management system that helps organizations manage employees, departments, salaries, and attendance efficiently.

## Features

- Employee Management
- Department Management
- Salary Management
- Dashboard Analytics
- Role-Based Authentication

## Tech Stack

### Frontend
- React.js
- Tailwind CSS
- Axios

### Backend
- Node.js
- Express.js

### Database
- MongoDB

## Installation

```bash
git clone repository_link
```

```bash
npm install
```

```bash
npm run dev
```
